# Elector Search Portal

A simple tool designed to assist Booth Level Officers (BLOs) and voters in locating their electoral records within the latest SIR (Special Summary Revision) data.

## Features

- **Search Functionality**: Find voters by Name, Relative's Name, and Assembly Constituency.
- **Fuzzy Search**: Automatically tries name variations to help find records even with slight spelling differences.
- **Multi-Assembly Search**: Search across multiple assemblies simultaneously.
- **Responsive Design**: Works seamlessly on desktop and mobile devices.

## Disclaimer

**This application is not officially affiliated with any government body.**

It is a helper tool created to facilitate easier searching of public electoral roll data. For official purposes, please always verify with official Election Commission documents and websites.
